# 自动微分

> 环境：pytorch + ipython

基础代码：

```python
In [9]: x = torch.arange(4.0)

In [10]: x
Out[10]: tensor([0., 1., 2., 3.])

In [11]: x.requires_grad_(True)
Out[11]: tensor([0., 1., 2., 3.], requires_grad=True)

In [12]: x.grad

In [13]: y = 2 * torch.dot(x,x)

In [14]: y
Out[14]: tensor(28., grad_fn=<MulBackward0>)

In [15]: y.backward()

In [16]: x.grad
Out[16]: tensor([ 0.,  4.,  8., 12.])
```
