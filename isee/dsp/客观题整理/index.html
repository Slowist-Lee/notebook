    5. 某系统的单位抽样响应为 \(h(n) = \delta(n) + 2\delta(n-1) + u(n-2) - u(n-3)\)，频率响应为 \(H(e^{j\omega})\)，则该系统的相位函数 \(\phi(\omega) = \_\_\_\_\_\_\_\_\)，零频率处的频率响应 \(H(e^{j0}) = \_\_\_\_\_\_\_\_\)。(来源: 2024A.docx_p12.jpg, 2024A.docx_p10.jpg)

    第一章 答案与解析

    一、判断题

    1. 答案: ×
      解析: 模拟周期信号 \(x_a(t)\) 的周期为 \(T_p\)，采样周期为 \(T_s\)。采样后得到的离散序列 \(x(n) = x_a(nT_s)\) 是周期的 当且仅当 \(T_p / T_s\) 为有理数。如果二者之比是无理数，则采样后序列为非周期序列。因此，该论断错误。

    2. 答案: ×
      解析: Z变换与拉普拉斯变换的关系是 \(z = e^{sT}\)。当 \(s=0\)（S平面原点）时，\(z = e^0 = 1\)，即Z平面的 \((1, 0)\) 点。因此，S平面原点对应Z平面上的 \(z=1\) 点，而不是Z平面原点。

    3. 答案: √
      解析: 系统函数 \(H(z)\) 的极点决定了系统的频率响应。极点靠近单位圆会产生峰值。如果极点集中在靠近 \(z=1\) 的实轴附近，意味着系统对低频分量（\(\omega \approx 0\)）有较大的响应，这正是低通滤波器的特性。

    4. 答案: ×
      解析: 时不变系统要求系统特性不随时间变化，即对任意延时 \(n_0\)，若输入为 \(x(n-n_0)\)，则输出为 \(y(n-n_0)\)。
      对于系统 \(y(n) = T[x(n)] = x(2n-3)+3\)：
      - 将输出延时 \(n_0\) 得：\(y(n-n_0) = x(2(n-n_0)-3)+3 = x(2n-2n_0-3)+3\)。
      - 将输入延时 \(n_0\) 得 \(x'(n) = x(n-n_0)\)，此时输出为 \(T[x'(n)] = x'(2n-3)+3 = x(2n-3-n_0)+3\)。
      因为 \(y(n-n_0) \neq T[x(n-n_0)]\)，所以该系统是时变系统。

    5. 答案: √
      解析: 设零点为 \(z_0 = r e^{j\omega_0}\)，其中 \(r \approx 1\)。系统频率响应 \(H(e^{j\omega})\) 是在单位圆上（\(z=e^{j\omega}\)）评估 \(H(z)\) 的结果。当 \(\omega=\omega_0\) 时，单位圆上的点 \(e^{j\omega_0}\) 与零点 \(z_0\) 非常接近，因此 \(|H(e^{j\omega_0})|\) 的值会非常小，从而在频谱上形成一个低谷。

    6. 答案: √
      解析: 抽取器 \(y(n) = x(nD)\) 是一个线性时变系统。线性易证。时变性：输入信号的移位 \(x(n-n_0)\) 变为输出 \(x(nD-n_0)\)，而输出信号的移位是 \(y(n-n_0) = x((n-n_0)D) = x(nD-n_0D)\)。两者不相等，所以是时变（移变）系统。题目陈述正确。

    二、选择题

    1. 答案:
      * (A), (B) 是周期信号, 周期分别是 5, 38。
      * (C), (D) 是非周期信号。
      解析:
      * (A) \(e^{j2\pi n/5}\): 数字角频率 \(\omega_0 = 2\pi/5\)。周期 \(N = 2\pi m / \omega_0 = 5m\)。取 \(m=1\) 得最小正整数周期 \(N=5\)。
      * (B) \(\sin(\pi n/19) = \sin(2\pi n/38)\): 数字角频率 \(\omega_0 = \pi/19\)。周期 \(N = 2\pi m / \omega_0 = 38m\)。取 \(m=1\) 得最小正整数周期 \(N=38\)。
      * (C) \(ne^{j\pi n}\): 幅度项 \(n\) 随时间变化，不满足周期性定义 \(x(n) = x(n+N)\)。
      * (D) \(e^{jn}\): 数字角频率 \(\omega_0 = 1\)。周期 \(N = 2\pi m / \omega_0 = 2\pi m\)。因为不存在整数 \(m\) 使得 \(N\) 为整数，所以是非周期的。

    2. 答案: B
      解析: 因果性（Causality）指的是系统在时刻 \(n\) 的输出只取决于当前和过去的输入（即 \(k \le n\)）。对于LTI（线性时不变）系统，这等价于其单位脉冲响应 \(h[n]\) 必须满足对于所有的 \(n<0\)，\(h[n]=0\)。

    3. 答案: A
      解析: LTI系统稳定的充要条件是其单位脉冲响应 \(h[n]\) 绝对可和，即 \(\sum_{n=-\infty}^{\infty}|h[n]| < \infty\)。这个条件也确保了 \(h[n]\) 的离散时间傅里叶变换（DTFT）存在且收敛。DTFT就是Z变换 \(H(z)\) 在单位圆上的取值。因此，如果 \(H(z)\) 的收敛域（ROC）包含单位圆，意味着DTFT存在，系统是稳定的。

    4. 答案: D
      解析: 两个有限长序列 \(x(n)\) 和 \(h(n)\) 进行线性卷积，若 \(x(n)\) 的长度为 \(M\)， \(h(n)\) 的长度为 \(N\)，则卷积结果 \(y(n) = x(n) * h(n)\) 的长度为 \(L = M + N - 1\)。

    5. 答案: B
      解析:
      * A: \(y(n) = 2nx(n)\)，系统参数 \(2n\) 随时间 \(n\) 变化，这是一个时变系统，不满足线性时不变（LTI）条件。
      * B: \(y(n) = x(n) + 0.5y(n-1)\) 是一个一阶递归系统，其系数为常数。可以写出其差分方程 \(y(n) - 0.5y(n-1) = x(n)\)，是典型的线性时不变系统。

    三、填空题

    1. 答案: \(\{3,2,1,4\}\)
      解析: \(x((n+1))_4\) 表示对序列 \(x(n)\) 先进行时间平移 \(n \to n+1\)（即左移1位），然后进行模4运算。
      \(x(n) = \{x(0), x(1), x(2), x(3)\} = \{4,3,2,1\}\)。
      左移1位相当于把序列的第一个元素移到最后，得到新序列 \(x'(n) = \{x(1), x(2), x(3), x(0)\} = \{3,2,1,4\}\)。

    2. 答案: 40
      解析: 信号周期是各分量周期的最小公倍数。
      - 对于 \(\cos(0.15\pi n) = \cos(2\pi \frac{3}{40} n)\)，其周期为 \(N_1 = 40/3\)。
      - 对于 \(2\sin(0.25\pi n) = 2\sin(2\pi \frac{1}{8} n)\)，其周期为 \(N_2 = 8\)。
      - 总周期 \(N\) 需要满足 \(N = k_1 N_1 = k_2 N_2\)，即 \(N = k_1 (40/3) = k_2 (8)\)，其中 \(k_1, k_2\) 为整数。为使 \(N\) 为整数， \(k_1\) 必须是3的倍数。取 \(k_1=3\)，则 \(N=40\)。此时 \(k_2=5\)。所以最小公共周期为40。

    3. 答案: \(\{3,4,5,1,2,0\}\) 或 \(\{3,4,5,1,2\}\) （取决于对原序列的理解）
      解析: \(x((n+2))_6\) 表示左移2位后模6。原序列 \(x(n) = \{1,2,3,4,5\}\)，长度为5。
      - 如果将其看作一个6点序列 \(\{1,2,3,4,5,0\}\)，则左移2位后得到 \(\{3,4,5,0,1,2\}\)。
      - 如果是在原序列长度5上进行循环移位，即 \(x((n+2))_5\)，则结果是 \(\{3,4,5,1,2\}\)。题目中的下标6通常表示运算的周期。

    4. 答案: \(\frac{\pi}{5T}\)
      解析: 数字角频率 \(\omega\) 和模拟角频率 \(\Omega\) 的关系是 \(\omega = \Omega T\)。
      因此，\(\Omega = \omega / T\)。
      当 \(\omega = \pi/5\) 时，对应的模拟频率是 \(\Omega = (\pi/5) / T = \frac{\pi}{5T}\)。

    5. 答案: \(\phi(\omega) = -\omega\), \(H(e^{j0}) = 4\)
      解析:
      首先简化 \(h(n)\)：
      \(h(n) = \delta(n) + 2\delta(n-1) + u(n-2) - u(n-3) = \delta(n) + 2\delta(n-1) + \delta(n-2)\)。
      所以这是一个FIR序列 \(h(n)=\{1, 2, 1\}\)。
      其频率响应为：
      $$
      H(e^{j\omega}) = \sum_{n=-\infty}^{\infty} h(n)e^{-j\omega n} = 1 + 2e^{-j\omega} + e^{-j2\omega}
      $$
      提出相位项 \(e^{-j\omega}\):
      $$
      H(e^{j\omega}) = e^{-j\omega} (e^{j\omega} + 2 + e^{-j\omega}) = e^{-j\omega} (2 + 2\cos(\omega))
      $$
      这是一个典型的广义线性相位形式 \(H(e^{j\omega}) = A(\omega)e^{j\phi(\omega)}\)。
      幅度响应 \(A(\omega) = 2 + 2\cos(\omega)\) (注意，当此值为负时，会贡献\(\pi\)的相位)。
      相位响应 \(\phi(\omega) = -\omega\)。
      在零频率处 (\(\omega=0\))，\(H(e^{j0}) = 1 + 2e^{0} + e^{0} = 1 + 2 + 1 = 4\)。

    第二章 答案与解析

    一、判断题

    1. 答案: √
      解析: 在数据后面补零（zero-padding）会增加DFT的点数N，这使得频域的采样点数增多，谱线间隔 \(\Delta f = f_s/N\) 变小。这相当于在原始信号的DTFT（离散时间傅里叶变换）频谱包络上进行了更密集的采样，得到的DFT频谱曲线会更平滑，有助于更精确地观察谱峰位置。因此，它提高了DFT的计算分辨率或谱线密度，但不能提高区分两个相近物理频率的能力（物理分辨率，由窗函数主瓣宽度决定）。在“提高频谱分辨率”这个表述下，通常理解为前者，故为正确。

    2. 答案: √
      解析: 对无限长信号进行DFT分析，必须先对其进行截断（加窗），这必然导致频谱泄漏（Leakage），即能量从原始频率扩散到相邻频率点。同时，DFT本身是在频域的离散采样，我们只能观察到这些采样点上的值，而可能错过采样点之间的真实谱峰，这就是栅栏效应（Picket-fence Effect）。这两个现象是DFT分析有限长度数据的固有问题。

    3. 答案: ×
      解析: 频谱混叠（Aliasing）是在模拟信号采样过程中，若采样频率不满足奈奎斯特采样定理（\(f_s \ge 2f_{max}\)），高频分量会混叠到低频范围。这是一个采样问题，不是DFT特有的。频谱泄漏是由于信号截断（加窗）引起的，是DFT进行频谱分析时的一个关键现象，但根源在于时域截断。

    4. 答案: √
      解析: 无论是按时间抽选（DIT）的基2-FFT算法，还是按频率抽选（DIF）的基2-FFT算法，其核心思想都是将一个N点DFT分解为两个N/2点的DFT。两种算法的计算复杂度完全相同，复数乘法次数均为 \((N/2)\log_2N\)，复数加法次数均为 \(N\log_2N\)。

    二、选择题

    1. 答案: D
      解析: 使用FFT计算线性卷积，是为了避免时域的循环卷积效应。线性卷积结果的长度为 \(L = N_1 + N_2 - 1\)。因此，用于计算的FFT点数 \(N\) 必须大于等于线性卷积的长度，即 \(N \ge L = N_1 + N_2 - 1\)。选项中，只有 \(N_1+N_2-1\) 是满足此条件的最小值。

    2. 答案: D
      解析: DFT（离散傅里叶变换）是DFS（离散傅里叶级数）的主值序列。DFS是处理时域周期、离散序列的，其变换结果也是频域周期、离散的。因此，DFT本质上处理的是一个时域周期、频域也周期的信号。

    3. 答案: D
      解析: 这是利用DFT/FFT计算线性卷积的标准流程。为了让循环卷积的结果等于线性卷积，必须对原始序列补零，使得变换长度 \(L\) 不小于线性卷积结果的长度 \(N+M-1\)。然后对补零后的序列做DFT，频域相乘，再做IDFT。

    4. 答案: B
      解析: 频谱分辨率（谱线间隔）\(\Delta f\) 由采样率 \(f_s\) 和DFT点数 \(N\) 决定：\(\Delta f = f_s / N\)。
      根据题意，\(\Delta f \le 0.5 \text{ Hz}\)，且 \(f_s = 2000 \text{ Hz}\)。
      所以，\(2000 / N \le 0.5 \implies N \ge 2000 / 0.5 = 4000\)。最少的DFT点数是4000。

    5. 答案: C
      解析:
      A. FFT（快速傅里叶变换）不是一种新的变换，而是计算DFT的一种高效算法。
      B. DFT的直接计算需要 \(N^2\) 次复数乘法。FFT需要 \((N/2)\log_2N\) 次。如果只计算单个 \(k\) 值的频谱 \(X(k)\)，直接计算需要 \(N\) 次复乘，而FFT需要计算所有值，所以此时直接计算更快。因此该选项错误。
      C. 基2-FFT算法要求序列长度 \(N\) 是2的整数次幂。如果不是，必须在序列末尾补零凑成 \(2^k\) 的长度。

    6. 答案: A
      解析: 线性卷积的长度为 \(L = 10 + 4 - 1 = 13\)。DFT的长度 \(N\) 必须满足 \(N \ge 13\)。选项中，A, B都是有效的长度。题目可能要求选择最小的有效长度，那么是13。如果选项中没有13，那么14, 15都是可以的。但如果这是个单选题，题目可能出得不严谨。如果D选项是13，那么D是最佳答案。在给定的选项中，A=14, B=15, D=12, C=13，那么C是正确答案。若选项如题所示，则A, B均可，A是大于等于13的最小选项。我们假定题目原意是C=13。

    7. 答案: C
      解析: 频域内两个序列的DFT的乘积，在时域对应的是这两个序列的循环卷积，而不是线性卷积。这是利用DFT计算线性卷积需要补零的根本原因。

    8. 答案: B
      解析: 这是DFT的频移特性（也叫调制特性）。一个序列 \(x(n)\) 乘以一个复指数序列 \(W_N^{-nk_0} = e^{-j\frac{2\pi}{N}nk_0}\)，其DFT结果是原序列的DFT \(X(k)\) 发生循环位移，即 \(X((k-k_0))_N\)。

    三、填空题

    1. 答案: 无限长非周期
      解析: DTFT（离散时间傅里叶变换）的定义适用于绝对可和的无限长序列，其结果是连续的频谱。DFT是为有限长序列定义的。因此，无限长非周期序列有DTFT但不能直接计算DFT（需要截断）。

    2. 答案: N
      解析: 一个N点序列的DFT结果也是一个N点序列，这N个点代表了在 \([0, 2\pi)\) 范围内的N个等间隔的频率分量。整个数字频率周期 \(2\pi\) 被划分成了N个频分量（或称频点、频箱）。

    3. 答案: \((N/2)\log_2N\), \(N\log_2N\)
      解析: 这是基2-FFT算法的标准计算复杂度。

    4. 答案: 主值序列 (或 一周期)，周期延拓
      解析: DFT与DFS在数值上是等价的。一个N点有限长序列的DFT，可以看作是将其进行周期延拓（周期为N）后得到的周期序列的一个周期的DFS。反之，该有限长序列就是其周期延拓序列的主值序列。

    5. 答案: 22, 800
      解析:
      * 根据DFT的定义 \(X[k] = \sum_{n=0}^{N-1} x[n]e^{-j\frac{2\pi}{N}nk}\)。当 \(k=0\) 时，\(X = \sum_{n=0}^{N-1} x[n]\)。所以 \(\sum X[k]\) 无法直接求，但题目可能问的是 \(X\)。如果是 \(X\)，则 \(X = 2+1+1+0+3+2+0+3+4+6 = 22\)。而 \(\sum_{k=0}^{9} X[k]\) 并不等于 \(X\)。但根据DFT反变换 \(x[n] = \frac{1}{N}\sum_{k=0}^{N-1}X[k]W_N^{-nk}\)，令n=0有 \(x = \frac{1}{N}\sum_{k=0}^{N-1}X[k]\)，所以 \(\sum_{k=0}^{9}X[k] = N \cdot x = 10 \times 2 = 20\)。 (注：第3章小测试题_p4.jpg中答案为20，说明题目是求和)。
      * 根据帕斯瓦尔定理（Parseval's Theorem）：\(\sum_{n=0}^{N-1} |x[n]|^2 = \frac{1}{N} \sum_{k=0}^{N-1} |X[k]|^2\)。
      所以，\(\sum_{k=0}^{N-1} |X[k]|^2 = N \sum_{n=0}^{N-1} |x[n]|^2\)。
      \(\sum |x[n]|^2 = 2^2+1^2+1^2+0^2+3^2+2^2+0^2+3^2+4^2+6^2 = 4+1+1+9+4+9+16+36 = 80\)。
      \(\sum |X[k]|^2 = 10 \times 80 = 800\)。

    6. 答案: \(-1.7-5.2j\)
      解析: 对于实序列 \(x(n)\)，其DFT具有共轭对称性：\(X[k] = X^*[N-k]\)，其中 \(*\) 表示共轭。
      本题中 \(N=9\)。
      \(X = X^*[9-5] = X^*\)。
      已知 \(X = -1.7+5.2j\)，其共轭为 \(X^* = -1.7-5.2j\)。
      所以，\(X = -1.7-5.2j\)。

      解析: 对于实系数的FIR滤波器，零点必须共轭成对出现。所以如果 \(z_1 = 0.5+0.5j\) 是一个零点，那么 \(z_2 = z_1^* = 0.5-0.5j\) 也必须是零点。

      \(z_3 = 1/z_1 = \frac{1}{0.5+0.5j} = \frac{0.5-0.5j}{0.5^2+0.5^2} = \frac{0.5-0.5j}{0.5} = 1-j\)。
      \(z_4 = 1/z_2 = (1/z_1)^* = (1-j)^* = 1+j\)。
      所以另外三个零点是 \(0.5-0.5j\), \(1-j\), \(1+j\)。

    5. 答案: 线性相位
      解析: 对于具有对称系数的线性相位FIR滤波器，可以利用其系数对称性 \(h(n)=h(N-1-n)\)，将输入信号先相加再做一次乘法，即 \(h(n)[x(n)+x(N-1-n)]\)。这样可以将乘法次数减少近一半，是最高效的结构。

    6. 答案: \(\frac{\sqrt{2}}{2}(1-j)\), \(\frac{\sqrt{2}}{2}(1+j)\), \(\frac{\sqrt{2}}{2}(1-j)\)
      解析:
      设零点 \(z_1 = \frac{\sqrt{2}}{2}(1+j) = e^{j\pi/4}\)。这是一个在单位圆上的零点。
      1. 共轭零点: 因为是实系数系统，共轭 \(z_2 = z_1^* = \frac{\sqrt{2}}{2}(1-j) = e^{-j\pi/4}\) 也必须是零点。
      2. 倒数零点: 因为是线性相位系统，倒数 \(z_3 = 1/z_1 = 1/e^{j\pi/4} = e^{-j\pi/4}\) 也必须是零点。这和 \(z_2\) 是同一个。
      3. 共轭倒数零点: \(z_4 = 1/z_2 = 1/e^{-j\pi/4} = e^{j\pi/4}\) 也必须是零点。这和 \(z_1\) 是同一个。
      这说明如果零点在单位圆上，则它的共轭点也必须存在，但倒数点就是其共轭点。这样只产生了2个零点。一个4阶系统应该有4个零点。这说明题目可能有隐含条件或错误。
      * 重新思考：一个4阶（N=5）线性相位FIR滤波器，如果零点在单位圆上，它们必须成对（共轭）出现。\(z_1, z_2\) 是一对。为了构成4个零点，必须还有另外一对零点。题目只给了一个，这暗示了其他零点与这一个的关系。根据线性相位零点分布的“四重奏”规则（\(z, z^*, 1/z, 1/z^*\)），如果一个零点不在单位圆上或实轴上，它会带来另外三个。但如果零点在单位圆上，则 \(z=1/z^*\)，这四个点会缩并成两个（\(z, z^*\)）。如果零点在实轴上，则 \(z=z^*\)，缩并成两个（\(z, 1/z\)）。如果零点是\(\pm1\)，则 \(z=z^*=1/z=1/z^*\)，只有一个点。
      此题中 \(z_1\) 在单位圆上，所以 \(z_1, z_1^*\) 构成了一对零点。为了满足4阶，一定还有另一对零点 \(z_3, z_3^*\)。题目没有给出任何关于这对零点的信息。
      * 唯一的可能性是这是一个二重零点。即 \(z_1 = e^{j\pi/4}\) 是二重根，那么其共轭 \(z_2 = e^{-j\pi/4}\) 也必须是二重根。这样总共就有4个零点了。
      所以，其他三个零点是 \(e^{-j\pi/4}\) (共轭根)，以及重复的 \(e^{j\pi/4}\) 和 \(e^{-j\pi/4}\)。
      因此答案是: \(\frac{\sqrt{2}}{2}(1-j)\), \(\frac{\sqrt{2}}{2}(1+j)\), \(\frac{\sqrt{2}}{2}(1-j)\)。

    第六章 答案与解析

    一、判断题

    1. 答案: ×
      解析: 零输入极限环是IIR滤波器（递归系统）中，由于对乘法器输出进行量化（舍入或截断） 引入的非线性效应导致的。它是在反馈回路中，即使输入为零，输出也会维持一个小的非零振荡。系数量化是另一回事，它会改变滤波器的频率响应，但不是极限环的直接原因。极限环的直接原因是运算过程中的量化。

    2. 答案: √
      解析: 极限环的幅度与量化阶距 \(Q\) 有关，而量化阶距由字长决定（\(Q=2^{-B}\)，B为小数位数）。增加系数字长（通常意味着也增加了数据通路字长），使得量化阶距 \(Q\) 变小，因此由量化误差引起的极限环震荡的幅度也会相应减小。

    3. 答案: √
      解析: 线性相位是由冲激响应的对称性（\(h(n) = \pm h(N-1-n)\)）决定的。在直接型结构中，滤波器的系数就是 \(h(n)\) 的值。即使这些系数 \(h(n)\) 被量化，只要量化后的系数 \(h_Q(n)\) 仍然保持对称性（即 \(h_Q(n) = \pm h_Q(N-1-n)\)），那么系统的线性相位特性就能被保持。常规的量化方法（如舍入）可以维持这种对称性。

    二、选择题

    1. 答案: C
      解析:
      - 原码表示: 符号位为1表示负数，数值部分直接按权展开。\(1.1011_原 \implies -(1 \cdot 2^{-1} + 0 \cdot 2^{-2} + 1 \cdot 2^{-3} + 1 \cdot 2^{-4}) = -(0.5 + 0.125 + 0.0625) = -0.6875\)。
      - 补码表示: 符号位为1表示负数。其值等于 \(-1 \times \text{符号位权值} + \text{数值部分权值之和}\)。\(-1 \cdot 2^0 + 1 \cdot 2^{-1} + 0 \cdot 2^{-2} + 1 \cdot 2^{-3} + 1 \cdot 2^{-4} = -1 + 0.5 + 0.125 + 0.0625 = -1 + 0.6875 = -0.3125\)。
      所以选C。

    2. 答案: D
      解析:
      A. 这是正确的，字长每增加1bit，量化噪声功率降低约6dB，信噪比（SNR）增加约6dB。
      B. 直接型结构中，传输函数的分母系数直接由高阶多项式的系数决定，这些系数对极点位置（多项式的根）非常敏感。而级联型将多项式分解为一阶或二阶因子的乘积，每个二阶节的系数只影响一对极点，敏感度大大降低。所以该说法正确。
      C. 乘法运算会导致结果的字长加倍，必须进行量化才能存回寄存器，这个过程会引入运算噪声，其大小与结构有关。说法正确。
      D. 这是一个错误的说法。一个零点（或极点）的位置是传输函数分子（或分母）多项式所有系数共同作用的结果，改变任何一个系数都会影响到所有的零点（或极点）。

    3. 答案: C
      解析:
      A, B, D都是正确的陈述。
      C. 截尾（Truncation）产生的误差是有偏的（总是朝零的方向），其均值不为零。舍入（Rounding）产生的误差是无偏的，均值为零。它们的统计特性（如概率密度函数、均值、方差）是不同的。所以该说法错误。

    4. 答案: B, C, D
      解析: IIR滤波器的极点位置对频率响应影响极大。直接型结构对系数量化非常敏感，高阶时尤其严重，A错误。级联型和并联型结构将高阶系统分解为多个低阶（一阶或二阶）子系统的组合，大大降低了系数量化对极点位置的影响，B和C正确。增加系数的字长可以直接提高系数的表示精度，从而减小量化误差，D正确。

    5. 答案: B
      解析:
      - 理论系统 \(H(z) = \frac{z^{-1}}{1-0.4z^{-1}}\)，极点在 \(p=0.4\)。
      - 低频处对应 \(z \approx 1\)（\(\omega=0\)）。理论响应 \(H(1) = \frac{1}{1-0.4} = \frac{1}{0.6} \approx 1.667\)。
      - 系数 \(a_1 = -0.4\)。用4位原码表示（1位符号，3位小数）。
      - \(0.4_{10} = 0.011001..._2\)。
      - 用3位小数截尾，得到 \(0.011_2 = 0.5+0.25=0.25+0.125=0.375\)。
      - 所以量化后的系数 \(a_{1Q} = -0.375\)。
      - 实际系统 \(H_Q(z) = \frac{z^{-1}}{1-0.375z^{-1}}\)。
      - 实际低频响应 \(H_Q(1) = \frac{1}{1-0.375} = \frac{1}{0.625} = 1.6\)。
      - 因为 \(1.6 < 1.667\)，所以实际响应偏低。

    4. 答案: 偏低, 偏高
      解析:
      - 理论系统极点 \(p = -0.7702025\)。高频处对应 \(z \approx -1\)（\(\omega=\pi\)）。理论响应 \(H(-1) = \frac{-1}{1-0.7702025} \approx -4.35\)。

      - 实际高频响应 \(H_Q(-1) = \frac{-1}{1-0.75} = -4\)。
      - \(|-4| < |-4.35|\)，所以幅值偏低。
      - 补码编码:
      - 系数 \(a_1 = 0.7702025\) 是正数，其补码表示与原码相同，仍为 \(0.110_2 = 0.75\)。
      - 但题目可能意在考察负数。我们假设系数是 \(-0.7702025\)。
      - \(0.7702025 \to 0.110\) (截尾)。原码为 \(1.110\)。
      - 补码：\(1.001 + 1 = 1.010_补 = -1+0.25 = -0.75\)。
      - 如果系数是 \(-0.7702025\)，截尾后为 \(-0.75\)。极点从 \(-0.77..\) 移向0到 \(-0.75\)。
      - 让我们回到正系数 \(0.7702025\)。

      - 这个解释似乎更合理，符合出题意图。让我们检查一下原题，是 \(1+0.77...z^{-1}\)。
      - 极点 \(p = -0.77...\)
      - 原码截尾：\(-0.77... \to -0.75\) (向0移动)。极点从-0.77..移到-0.75。
      - 补码截尾：\(-0.77... \to -0.875\) (远离0移动)。极点从-0.77..移到-0.875。
      - 高频响应 \(H(-1) = \frac{-1}{1-p}\)。
      - 理论值：\(p=-0.77...\), \(H(-1)=\frac{-1}{1.77...} \approx -0.56\)。
      - 原码截尾：\(p_Q=-0.75\), \(H_Q(-1)=\frac{-1}{1.75} \approx -0.57\)。幅值 \(|-0.57| > |-0.56|\)，偏高。
      - 补码截尾：\(p_Q=-0.875\), \(H_Q(-1)=\frac{-1}{1.875} \approx -0.53\)。幅值 \(|-0.53| < |-0.56|\)，偏低。
      - 最终结论: 这个题目的符号给的是"+"，所以极点为负。因此原码截尾导致极点向0移动，实际响应偏高；补码截尾导致极点远离0移动，实际响应偏低。

    1. 假设序列 \(x(n)\) 的取样频率为1000Hz, 现对其进行每隔1个采样点抽取一个样点的2倍抽取处理, 所得序列的抽样频率为________。为保证抽取后不产生混叠失真, 原信号带宽必须限制在范围________。(来源: 2024A.docx_p10.jpg)

    第七章 答案与解析

    一、判断题

    1. 答案: √
      解析: 在时域对信号进行 \(I\) 倍内插（插入 \(I-1\) 个零点），在频域上的效果是频谱被压缩到原来的 \(1/I\)，并且在 \([0, 2\pi)\) 范围内产生 \(I-1\) 个重复的镜像频谱。这些镜像是失真，需要后续的低通滤波器来滤除。在没有经过滤波前，原始信号的频谱信息确实被这些镜像所“污染”，可以理解为一种失真。因此该陈述正确。

    2. 答案: √
      解析: 这是多抽样率处理中的一个基本原则。在抽取（降采样） 前，必须使用抗混叠滤波器滤掉高于新奈奎斯特频率的信号分量。这个滤波过程本身就是一种有意的高频信息损失，但这是为了防止混叠失真——一种更严重的、不可逆的错误，它会把高频信息错误地叠加到低频信息上。因此，宁可损失高频信息，也要避免混叠。

    二、选择题

    1. 答案: B
      解析:
      A. 内插是在原信号的样本点之间插入0，而不是插入原信号的样本值。错误。
      B. 时域插入0点，相当于将原序列在时间轴上“拉伸”了I倍（展宽）。根据傅里叶变换的时域/频域尺度变换特性，时域展宽对应频域的压缩和幅度增益。频谱被压缩为原来的 \(1/I\)，并在 \([0, 2\pi)\) 内重复I次，形成1个基带谱和I-1个镜像。正确。
      C. 为了消除镜像，需要接入低通滤波器。但为了恢复信号的原始幅度，滤波器的增益应该是 \(I\)，而不是1。错误。
      D. 镜像失真是内插（升采样）的产物，混叠失真是抽取（降采样）或采样不当的产物。两者机理和表现都不同。错误。

    2. 答案: C
      解析:
      A. 通过 D/A -> 模拟滤波器 -> A/D 的方式可以实现任意采样率转换，但这个过程中的 D/A 和 A/D 都会引入误差。正确。
      B. 抽取是从原始信号中每D个点取一个，是下采样，可以减少数据量。正确。
      C. 整数因子内插是在样本点之间插入I-1个零点，而不是采用零阶保持。零阶保持是D/A转换的一种方法，不是内插操作本身。错误。
      D. 有理数因子 \(I/D\) 转换，需要先I倍内插再D倍抽取。内插后的抗镜像滤波器和抽取前的抗混叠滤波器可以合并成一个单一的低通滤波器。正确。

    3. 答案: A
      解析:
      - 采样率从 \(24 \text{ kHz}\) 变为 \(15 \text{ kHz}\)，是降低采样率，属于降采样。
      - 转换因子为 \(15/24 = 5/8\)。这意味着需要先进行5倍内插，再进行8倍抽取 (\(I=5, D=8\))。
      A. 最终结果是降采样，为了防止抽取时的混叠，必须用低通滤波器滤除高于新奈奎斯特频率（\(15/2=7.5 \text{ kHz}\)）的信号，这必然导致原来在 \(7.5 \text{ kHz}\) 到 \(12 \text{ kHz}\) 之间的信息丢失。正确。
      B. 滤波器的截止频率应该设置在 \(\min(\frac{f_s}{2}, \frac{f_{s,new}}{2})\) 处，即 \(7.5 \text{ kHz}\)。相对于原始24kHz采样率，其归一化频率为 \(\omega_c = 2\pi \frac{7.5}{24} = \frac{5}{8}\pi\)。或者，相对于内插后的 \(120 \text{ kHz}\) 采样率，截止频率为 \(\omega_c = 2\pi \frac{7.5}{120} = \frac{\pi}{8}\)。选项中只给出了\(\pi/8\)，没有说明是相对于哪个采样率，但数值是正确的。然而，A选项的描述更具普遍性和根本性。
      C. 滤波器既有抗混叠（为抽取服务）的作用，也有抗镜像（为内插服务）的作用。
      D. 必须先内插，后抽取，以避免过早地丢失有用信息。错误。
      在所有选项中，A是最准确和核心的描述。

    三、填空题

    1. 答案: \(500 \text{ Hz}\), \([-250 \text{ Hz}, 250 \text{ Hz}]\)
      解析:
      - “每隔1个采样点抽取一个样点”意味着保留第0, 2, 4, ...个样本，丢弃第1, 3, 5, ...个样本。这是2倍抽取（Decimation by 2, \(D=2\)）。
      - 新的抽样频率 \(f_s' = f_s / D = 1000 \text{ Hz} / 2 = 500 \text{ Hz}\)。
      - 为了保证抽取后不发生混叠，原始信号的最高频率必须低于新的奈奎斯特频率 \(f_{N'} = f_s' / 2 = 500 \text{ Hz} / 2 = 250 \text{ Hz}\)。因此，原信号的带宽必须限制在 \([-250 \text{ Hz}, 250 \text{ Hz}]\) 范围内。 \([-250 \text{ Hz}, 250 \text{ Hz}]\) 范围内。

    3. 某线性相位且实系数的FIR滤波器系统有零点 \(z=-0.5j\), 以及 \(z=1\), 则该系统至少有________个零点。(来源: 复习：第5章 FIR数字滤波器设计和实现_p6.jpg)

    答案与解析

    一、判断题

    1. 答案: ×
      解析: 命题的前半部分是正确的：在满足相似的频率响应指标（特别是陡峭的过渡带）时，IIR滤波器的阶数通常远低于FIR滤波器。但后半部分的原因解释是错误的。IIR滤波器之所以高效，是因为它同时使用极点和零点来塑造频谱。靠近单位圆的极点可以以很低的阶数实现非常陡峭的响应。而FIR滤波器只有零点，需要更高阶的零点分布才能达到同样的效果。FIR滤波器的冲激响应对称性是其能够实现严格线性相位的原因，而不是其阶数较高的原因。

    2. 答案: √
      解析: FIR滤波器由于其结构中没有反馈环路（非递归），其冲激响应是有限长的，因此系统总是稳定的。这适用于直接型、级联型等绝大多数结构。但频率取样型结构是一个特例。它在内部使用了一组位于单位圆上的极点（谐振器）来生成频率样本，然后用一组零点去精确抵消这些极点。在理论上，零极点完全抵消，系统是FIR且稳定的。但在有限字长的实际硬件中，系数的量化会导致零点和极点的位置有微小偏差，无法完全抵消。这会使得单位圆上的极点“暴露”出来，导致系统不稳定。

    二、选择题

    1. 答案: A, D
      解析:
      * A: \(H(z)=1-z^{-1}\) 对应的冲激响应是 \(h(n)=\{\mathbf{1}, -1\}\) (在n=0, 1处取值)。这是一个长度N=2的序列，中心点在0.5。它满足反对称条件 \(h(n)=-h(N-1-n)\)，因为 \(h(0) = 1, h(1) = -1\), \(h(0)=-h(1)\)。这是IV型线性相位。
      * B: 极点呈互为倒数的共轭对（\(p\) 和 \(1/p^*\)）是全通滤波器的特性。全通滤波器幅度响应为常数，但相位响应是非线性的。
      * C: 系统的冲激响应是复数。具有线性相位的实系数滤波器，其冲激响应必须是实数。
      * D: 这是一个理想低通滤波器的冲激响应（sinc函数）被延迟了15个采样点。Sinc函数本身是关于原点偶对称的。延迟后的 \(h(n)\) 是关于 \(n=15\) 偶对称的。任何对该序列的对称截取（加窗）都会保持对称性，从而具有线性相位。

    2. 答案: (需根据图片判断)
      解析: 这类题的解法是：
      * 稳定性: 如果所有极点（×）都在单位圆内部，系统是稳定的。只要有任何一个极点在单位圆上或圆外，系统就是不稳定的。
      * 因果性: 对于有理系统函数，如果收敛域（ROC）在最外层极点的外部并包含 \(z=\infty\)，则系统是因果的。
      * 滤波器类型:
      * 低通: 极点靠近 \(z=1\) (\(\omega=0\))，零点靠近 \(z=-1\) (\(\omega=\pi\))
      * 高通: 极点靠近 \(z=-1\) (\(\omega=\pi\))，零点靠近 \(z=1\) (\(\omega=0\))
      * 带通: 极点在单位圆上除 \(z=\pm 1\) 之外的共轭位置，零点在 \(z=\pm 1\) 处。
      * 带阻: 零点在单位圆上除 \(z=\pm 1\) 之外的共轭位置，极点用于平滑通带。
      * 线性相位: 对于FIR滤波器，零点需要满足共轭倒数对称分布。

    三、填空题

    1. 答案: 无限长脉冲响应 (IIR)，有限长脉冲响应 (FIR)
      解析: 这是数字滤波器最基本的分类方式。如果系统的单位脉冲响应 \(h(n)\) 持续无限长时间，则称为IIR (Infinite Impulse Response) 滤波器，其结构中必然包含反馈。如果 \(h(n)\) 只在有限个点上非零，则称为FIR (Finite Impulse Response) 滤波器，其结构是前馈的。

    2. 答案: 级联型, 线性相位型
      解析:
      * IIR: 在级联型结构中，高阶系统被分解为一阶和二阶子系统的乘积。每个二阶子系统控制着一对零点和一对极点。通过调整某个子系统的分子系数，可以比较独立地调整该对零点的位置，对其他零极点影响较小。
      * FIR: 线性相位型结构利用了FIR滤波器冲激响应的对称性 (\(h(n)=\pm h(N-1-n)\))，将对称位置的输入先相加，然后进行一次乘法运算。这使得乘法器的数量可以从直接型的 \(N\) 个减少到大约 \(N/2\) 个，是乘法效率最高的结构。

    3. 答案: 5
      解析: 对于具有实系数和线性相位的FIR滤波器，其零点分布必须满足“四重奏”规则：如果 \(z_0\) 是一个零点，那么它的共轭 \(z_0^*\)、倒数 \(1/z_0\)、以及共轭倒数 \(1/z_0^*\) 都必须是零点。
      1. 从给定的零点 \(z_1 = -0.5j = 0.5e^{-j\pi/2}\) 开始：
      * 其共轭是 \(z_2 = z_1^* = 0.5j\)。
      * 其倒数是 \(z_3 = 1/z_1 = 1/(-0.5j) = 2j\)。
      * 其共轭倒数是 \(z_4 = z_3^* = -2j\)。
      这四个零点 \(\{0.5j, -0.5j, 2j, -2j\}\) 构成一个满足要求的集合。
      2. 从给定的零点 \(z_5 = 1\) 开始：
      * 它位于实轴上，所以 \(z_5 = z_5^*\)。
      * 它位于单位圆上，所以 \(z_5 = 1/z_5\)。
      因此，这个零点自身就满足了所有对称性要求，可以单独存在。
      3. 将这两组零点组合起来，得到的最少零点数量是 \(4 + 1 = 5\) 个。这对应一个5阶的FIR滤波器（长度N=6），其传输函数为 \(H(z) = C(z-0.5j)(z+0.5j)(z-2j)(z+2j)(z-1)\)，系数是实数且满足线性相位条件。
