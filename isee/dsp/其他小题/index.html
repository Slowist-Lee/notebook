    Chap1 离散时间信号与离散时间系统

    一、选择题

    1. 一个LTI系统的差分方程为 \(y(n) = 0.5x(n) - 0.2x(n-1) + 0.8y(n-1)\)，该系统属于 ( )
      A. FIR系统
      B. IIR系统
      C. 非因果系统
      D. 非线性系统

    2. 在Z平面上，若一个零点位于单位圆上 \(z=e^{j\omega_0}\) 处，则系统频率响应在 \(\omega = \omega_0\) 处的幅值 |H(ejω)| ( )
      A. 取得极大值
      B. 取得极小值
      C. 等于零
      D. 等于1

    3. 若一个右边序列 \(x(n)\) (即当 \(n<N_1\) 时，\(x(n)=0\) )的Z变换 \(X(z)\) 的极点为 \(p_1, p_2, ..., p_N\)，则其收敛域(ROC)为 ( )
      A. \(|z| < \min(|p_i|)\)
      B. \(|z| > \max(|p_i|)\)
      C. \(\min(|p_i|) < |z| < \max(|p_i|)\)
      D. 整个Z平面，除了原点

    4. 一个离散时间系统的输入输出关系为 \(y(n) = x(n) + 3\)，该系统是 ( )
      A. 线性时不变系统
      B. 线性时变系统
      C. 非线性时不变系统
      D. 非线性时变系统

    5. 已知离散时间傅里叶变换（DTFT）对 \(x(n)h(n) \leftrightarrow \frac{1}{2\pi}X(e^{j\omega})*H(e^{j\omega})\)，这个性质被称为 ( )
      A. 线性性质
      B. 频移性质
      C. 时域卷积定理
      D. 频域卷积定理 (或 时域相乘定理)

    6. 一个双边序列 \(x(n)\) 的Z变换 \(X(z)\) 的收敛域（ROC）通常是 ( )
      A. 一个圆盘
      B. 一个圆环
      C. 圆的外部
      D. Z平面

    二、填空题

    1. 将一个模拟信号转换为数字信号，需要经过三个基本步骤，分别是、和。

    2. 要使采样后的信号能不失真地恢复出原模拟信号，采样频率 \(f_s\) 必须大于或等于模拟信号最高频率 \(f_h\) 的两倍，这被称为定理，满足该条件的最低采样频率称为。

    3. 已知一个LTI系统的系统函数为 \(H(z) = \frac{z}{z-0.5}\)，且系统是因果的，则其收敛域为。

    4. 将模拟信号 \(x_a(t)\) 经过理想采样后得到离散信号 \(x(n)\)，其傅里叶变换 \(X(e^{j\omega})\) 是 \(x_a(t)\) 的傅里叶变换 \(X_a(j\Omega)\) 的。

    5. 离散时间傅里叶变换（DTFT）的频谱是的，其周期为。

    Chap1 答案与解析

    一、选择题

    1. 答案: B
      解析: 该系统的差分方程中，当前输出 \(y(n)\) 不仅与当前和过去的输入有关，还与过去的输出 \(y(n-1)\) 有关。这种带有反馈（递归）特性的系统是无限长冲激响应（IIR）系统。FIR（有限长冲激响应）系统的输出只与当前和过去的输入有关。

    2. 答案: C
      解析: 这是Z变换频率响应的几何解释。系统频率响应的幅值是Z平面上单位圆上的点到所有零点的距离之积，除以到所有极点的距离之积。当一个零点正好在单位圆上时，单位圆上的点扫到该零点位置时，其距离为0，因此整个频率响应的幅值在该频率点上为0。

    3. 答案: B
      解析: 这是Z变换收敛域的一个重要性质。对于一个右边序列（因果序列是其特例），其Z变换的收敛域位于最外侧极点的外部，即 \(|z|\) 大于所有极点模的最大值。

    4. 答案: C
      解析:
      * 线性判断：检验是否满足叠加原理。
      * 齐次性：令输入为 \(ax(n)\)，输出为 \(T[ax(n)] = ax(n) + 3\)。而 \(a \cdot y(n) = a(x(n)+3) = ax(n) + 3a\)。因为 \(T[ax(n)] \neq a \cdot y(n)\)，不满足齐次性，所以系统是非线性的。
      * 时不变性判断：检验输入移位是否等于输出移位。
      * 将输入延时 \(n_0\) 得 \(x'(n) = x(n-n_0)\)，输出为 \(y'(n) = x(n-n_0) + 3\)。
      * 将输出延时 \(n_0\) 得 \(y(n-n_0) = x(n-n_0) + 3\)。
      * 因为 \(y'(n) = y(n-n_0)\)，所以系统是时不变的。
      * 综上，该系统是非线性时不变系统。

    5. 答案: D
      解析: 这是傅里叶变换的一个重要对偶性质。时域中的卷积对应频域中的相乘（时域卷积定理），而时域中的相乘则对应频域中的周期卷积（频域卷积定理）。

    6. 答案: B
      解析: 根据Z变换收敛域的性质，一个双边序列（在 \(n>0\) 和 \(n<0\) 方向上都存在非零值）的收敛域是由其因果部分和反因果部分收敛域的交集构成的。因果部分的收敛域是某个圆的外部，反因果部分的收敛域是某个圆的内部，它们的交集通常是一个圆环。

    二、填空题

    1. 答案: 抽样 (或 采样)、量化、编码
      解析: 这是模拟信号数字化的标准流程。首先在时间轴上离散化（抽样），然后在幅度上离散化（量化），最后用二进制码表示量化后的值（编码）。

    2. 答案: 奈奎斯特采样 (Nyquist Sampling), 奈奎斯特频率 (Nyquist Rate)
      解析: \(f_s \ge 2f_h\) 是著名的奈奎斯特采样定理。最低采样频率 \(2f_h\) 被称为奈奎斯特频率。

    3. 答案: \(|z| > 0.5\)
      解析: 系统函数 \(H(z) = \frac{z}{z-0.5}\) 有一个极点在 \(p=0.5\)。对于因果系统，其收敛域是位于最外层极点之外的区域，因此收敛域为 \(|z| > 0.5\)。

    4. 答案: 周期延拓
      解析: 这是采样定理在频域的体现。对一个模拟信号进行时域采样，其效果是在频域上，将原始模拟信号的频谱以采样频率 \(\Omega_s\) 为周期进行周期性的延拓。

    5. 答案: 连续，\(2\pi\)
      解析: 离散时间傅里叶变换 \(X(e^{j\omega})\) 是关于数字角频率 \(\omega\) 的函数。由于 \(e^{j(\omega+2\pi)} = e^{j\omega}\)，所以DTFT是以 \(2\pi\) 为周期的周期函数。在每个周期内，\(\omega\) 是一个连续变量，因此频谱是连续的。

    Chap2 离散傅里叶变换及其快速计算方法

    一、选择题

    1. 一个长度为 \(N\) 的实数序列 \(x(n)\)，其 \(N\) 点DFT为 \(X(k)\)，则 \(X(k)\) 具有何种对称性 ( )
      A. \(X(k) = X(N-k)\)
      B. \(X(k) = -X(N-k)\)
      C. \(X(k) = X^*(N-k)\)
      D. \(X(k) = -X^*(N-k)\)

    二、填空题

    1. 在基2-DIT-FFT算法中，输入序列需要进行排序，而输出序列是按自然顺序排列的。

    2. 对一个时域离散、周期的信号进行傅里叶分析，所用的变换是，其频谱具有的特点。

    Chap2 答案与解析

    一、选择题

    1. 答案: C
      解析: 这是实序列DFT的一个重要性质：共轭对称性。对于实序列 \(x(n)\)，其DFT \(X(k)\) 满足 \(X(k) = X^*(N-k)\)，其中 * 表示复共轭。这意味着其实部是偶对称的，虚部是奇对称的。

    二、填空题

    1. 答案: 位倒序 (或 比特反转)
      解析: 在按时间抽选（DIT）的基2-FFT算法中，为了实现原位计算，输入序列 \(x(n)\) 的顺序需要按照其下标的二进制位倒序来重新排列。

    2. 答案: 离散傅里叶级数 (DFS), 离散、周期
      解析: 这是傅里叶变换家族的对偶关系。时域和频域的周期性/离散性是对应的。时域离散导致频域周期，时域周期导致频域离散。因此，一个时域离散且周期的信号，其傅里叶表示（DFS）在频域也是离散且周期的。

    Chap3 IIR滤波器设计

    一、选择题

    1. I型切比雪夫（Chebyshev I）滤波器的幅频响应特性是 ( )
      A. 通带和阻带都单调
      B. 通带等波纹，阻带单调
      C. 通带单调，阻带等波纹
      D. 通带和阻带都等波纹

    2. 在双线性变换法中，为了保证数字滤波器的截止频率 \(\omega_p\) 和设计指标完全一致，需要在设计前对模拟滤波器的对应频率指标进行 ( )
      A. 频率采样
      B. 频率预畸变 (pre-warping)
      C. 频率归一化
      D. 频率变换

    3. 与巴特沃斯滤波器相比，在满足相同阶数和截止频率的条件下，切比雪夫I型滤波器的主要特点是 ( )
      A. 通带和阻带都是平坦的
      B. 通带是等波纹的，而阻带是单调下降的
      C. 通带是单调的，而阻带是等波纹的
      D. 通带和阻带都是等波纹的

    4. 在设计IIR数字滤波器时，为了将归一化的模拟低通原型滤波器 \(H_{lp}(p)\) 变换为一个中心频率为 \(\Omega_0\)、带宽为 \(B\) 的模拟带通滤波器，应使用的频率变换是 ( )
      A. \(p = s/\Omega_p\)
      B. \(p = \Omega_p/s\)
      C. \(p = \frac{s^2+\Omega_0^2}{Bs}\)
      D. \(p = \frac{Bs}{s^2+\Omega_0^2}\)

    5. 在设计一个IIR数字低通滤波器时，若采用双线性变换法，并且设计指标给定了数字域的通带截止频率 \(\omega_p\) 和阻带起始频率 \(\omega_s\)，设计过程的第一步是 ( )
      A. 直接使用 \(\Omega_p = \omega_p/T\) 和 \(\Omega_s = \omega_s/T\)
      B. 进行频率预畸变，计算 \(\Omega_p = \frac{2}{T}\tan(\frac{\omega_p}{2})\) 和 \(\Omega_s = \frac{2}{T}\tan(\frac{\omega_s}{2})\)
      C. 直接用数字频率指标来选择滤波器阶数
      D. 确定模拟滤波器的类型为高通

    二、填空题

    1. IIR数字滤波器设计主要有两种经典方法，分别是法和法。

    2. 在巴特沃斯和切比雪夫滤波器设计中，决定滤波器过渡带陡峭程度的关键参数是。

    3. 切比雪夫I型滤波器的幅度平方函数为 \(|H(j\Omega)|^2 = \frac{1}{1+\epsilon^2 C_N^2(\Omega/\Omega_p)}\)，其中参数 \(\epsilon\) 控制着的幅度。

    4. 对于一个模拟低通巴特沃斯滤波器，其幅度平方函数为 \(|H(j\Omega)|^2 = \frac{1}{1+(\Omega/\Omega_c)^{2N}}\)。在通带截止频率 \(\Omega_p\) 处，其衰减为 \(A_p\) (dB)，则 \(|H(j\Omega_p)|^2\) 与 \(A_p\) 的关系为 $|H(j\Omega_p)|^2 = $ 。

    Chap3 答案与解析

    一、选择题

    1. 答案: B
      解析: 这是不同模拟滤波器原型的标志性特点。
      * 巴特沃斯（Butterworth）滤波器在通带和阻带都是最大平坦（maximally flat），即单调变化。
      * I型切比雪夫（Chebyshev I）滤波器在通带是等波纹的，而在阻带是单调的。
      * II型切比雪夫（Chebyshev II / Inverse Chebyshev）滤波器在通带是单调的，而在阻带是等波纹的。
      * 椭圆（Elliptic / Cauer）滤波器在通带和阻带都是等波纹的。

    2. 答案: B
      解析: 双线性变换法引入了模拟频率 \(\Omega\) 和数字频率 \(\omega\) 之间的非线性关系 \(\Omega = \frac{2}{T}\tan(\frac{\omega}{2})\)。为了确保最终的数字滤波器在关键频率点（如通带截止频率、阻带起始频率）上满足设计指标，必须先将数字域的这些频率指标通过这个非线性关系反向映射到模拟域，得到一组“预畸变”的模拟频率指标，然后再用这些预畸变的指标去设计模拟滤波器。这个过程称为频率预畸变。

    3. 答案: B
      解析: 这是切比雪夫I型滤波器的定义特性。它通过允许通带内存在等幅度的波纹（equiripple），来换取比同阶巴特沃斯滤波器更陡峭的过渡带特性。其阻带响应仍然是单调下降的。相比之下，巴特沃斯在通带和阻带都是“最大平坦”的（maximally flat），即单调。

    4. 答案: C
      解析: 这是标准的模拟滤波器频率变换（频带变换）公式。将低通原型中的变量 \(p\) 替换为表达式 \(\frac{s^2+\Omega_0^2}{Bs}\)，即可将低通特性映射为带通特性。选项A是低通到低通，B是低通到高通，D是低通到带阻的变换。

    5. 答案: B
      解析: 双线性变换法引入了数字频率 \(\omega\) 和模拟频率 \(\Omega\) 之间的非线性关系。为了确保设计出的数字滤波器在关键频率点（如 \(\omega_p, \omega_s\)）上满足指标，必须首先对这些数字频率指标进行“预畸变”（pre-warping），将其转换到模拟域对应的频率点 \(\Omega_p, \Omega_s\) 上，然后再进行后续的模拟滤波器设计。

    二、填空题

    1. 答案: 冲激响应不变法 (Impulse Invariance Method), 双线性变换法 (Bilinear Transform Method)
      解析: 这两种方法是从一个已知的模拟滤波器系统函数 \(H_a(s)\) 得到数字滤波器系统函数 \(H(z)\) 的最主要和最经典的方法。

    2. 答案: 滤波器的阶数 (N)
      解析: 对于这两种经典的模拟滤波器原型，阶数 \(N\) 越高，其频率响应在通带和阻带之间的过渡就越陡峭，即过渡带越窄。

    3. 答案: 通带波纹 (passband ripple)
      解析: 在切比雪夫I型滤波器的幅度平方函数表达式中，\(\epsilon\) (epsilon) 是一个决定通带波纹大小的参数。通带最大衰减 \(A_p\) (dB) 与 \(\epsilon\) 的关系为 \(A_p = 10\log_{10}(1+\epsilon^2)\)。\(\epsilon\) 越大，通带内的起伏就越大。

    4. 答案: \(10^{-A_p/10}\)
      解析: 衰减的定义是 \(A_p = 10\log_{10}(1/|H(j\Omega_p)|^2)\)，因此可以推导出 \(|H(j\Omega_p)|^2 = 10^{-A_p/10}\)。

    Chap4 FIR滤波器设计

    一、选择题

    1. 在窗函数法设计FIR滤波器时，若希望获得较小的过渡带宽度和较差的阻带衰减，应选择 ( )
      A. 矩形窗
      B. 汉宁窗
      C. 汉明窗
      D. 布莱克曼窗

    2. 某线性相位FIR滤波器的单位冲激响应为 \(h(n)\)，长度 \(N=11\)，且满足 \(h(n) = -h(10-n)\)。该滤波器属于哪种类型？ ( )
      A. Type I (偶对称，N奇)
      B. Type II (偶对称，N偶)
      C. Type III (奇对称，N奇)
      D. Type IV (奇对称，N偶)

    3. 在设计高通滤波器时，不应选用下列哪种类型的线性相位FIR滤波器？ ( )
      A. Type I
      B. Type II
      C. Type III
      D. Type IV

    4. 在窗函数法设计FIR滤波器中，固定窗函数类型（如汉明窗），若增加窗的长度N，将会导致 ( )
      A. 过渡带变宽，阻带衰减变好
      B. 过渡带变窄，阻带衰减变差
      C. 过渡带变宽，阻带衰减不变
      D. 过渡带变窄，阻带衰减基本不变

    二、填空题

    1. 线性相位FIR滤波器根据其冲激响应的对称性（偶对称/奇对称）和长度（奇/偶）可以分为四种类型。其中，频率响应在 \(\omega=0\) 和 \(\omega=\pi\) 处都必定为零的是型。

    2. 在频率采样法设计FIR滤波器时，通过在理想的通带采样点和阻带采样点之间，设置一些幅值介于0和1之间的采样点，这些点被称为，其作用是。

    Chap4 答案与解析

    一、选择题

    1. 答案: A
      解析: 窗函数的性能存在一个固有的折中：主瓣宽度和旁瓣衰减。主瓣越窄，过渡带越窄；旁瓣衰减越大，阻带衰减越大。在所有常用窗函数中，矩形窗的主瓣最窄（过渡带最窄），但其第一旁瓣相对最高（阻带衰减最差，约-13dB）。布莱克曼窗则相反，主瓣最宽，旁瓣衰减最大。

    2. 答案: C
      解析: 题目条件 \(h(n) = -h(10-n)\) 指明了冲激响应是奇对称（反对称）的。长度 \(N=11\) 是奇数。根据线性相位FIR滤波器的分类，奇对称且N为奇数的滤波器是Type III。

    3. 答案: B
      解析: Type II 线性相位FIR滤波器（偶对称，N偶）的频率响应幅值在 \(\omega=\pi\) 处恒为零。高通滤波器要求在接近 \(\pi\) 的高频区域有较大的增益，因此Type II滤波器无法满足高通滤波器的设计要求。

    4. 答案: D
      解析: 这是窗函数的一个核心特性。
      * 过渡带宽度与窗函数频谱的主瓣宽度成正比，而主瓣宽度与窗长 \(N\) 成反比。因此，增加 \(N\) 会使主瓣变窄，从而使过渡带变窄。
      * 阻带衰减取决于窗函数频谱的旁瓣相对幅度。对于一个特定类型的窗函数（如汉明窗），其旁瓣的相对幅度是基本固定的，不随窗长 \(N\) 的变化而显著变化。因此，阻带衰减基本不变。

    二、填空题

    1. 答案: Type III
      解析:
      * 奇对称（Type III 和 Type IV）的滤波器，由于其对称性，在 \(\omega=0\) 处的频率响应为零。
      * 奇对称且 \(N\) 为奇数（Type III）的滤波器，其频率响应在 \(\omega=\pi\) 处也为零。
      * 因此，Type III是在 \(\omega=0\) 和 \(\omega=\pi\) 处都有零点的类型。

    2. 答案: 过渡点 (或 过渡采样点), 增大阻带衰减
      解析: 这是频率采样法相对于简单理想采样的一种优化。在不连续点（通带到阻带的跳变处）附近设置过渡点，可以使内插后的频率响应更加平滑地从通带过渡到阻带，有效抑制由吉布斯现象引起的波纹，从而获得更大的阻带衰减。

    Chap6 答案与解析

    一、填空题

    1. 答案: \(\pm \frac{Q/2}{1-|a|}\)
      解析: 这是由零输入极限环振荡分析得出的死区范围。当输出值 \(y(n-1)\) 小到使其与系数 \(a\) 的乘积的绝对值小于半个量化阶距（\(Q/2\)）时，乘积结果被舍入为零，系统状态不再改变，形成极限环。这个边界值就是死区的范围。

    一、选择题

    1. 在多抽样率信号处理中，为了提高计算效率，通常将滤波器进行多相分解。这一操作的主要目的是 ( )
      A. 使得滤波器阶数降低
      B. 使得滤波器具有线性相位
      C. 将运算从高采样率部分移动到低采样率部分
      D. 消除系统的非线性失真

    二、填空题

    1. 在高效抽取器结构中，抗混叠滤波器 \(H(z)\) 和抽取器 \(\downarrow D\) 的位置可以交换，但滤波器需要改变为 \(H(z^D)\)。这一等效变换被称为。

    Chap7 答案与解析

    一、选择题

    1. 答案: C
      解析: 多相分解的核心思想是将一个工作在高采样率 \(f_s\) 下的滤波器 \(H(z)\)，分解成多个并行的、工作在低采样率 \(f_s/D\) 或 \(f_s/I\) 下的子滤波器（多相元件）。通过与抽取/内插操作的顺序交换（利用Noble恒等式），使得大量的滤波运算在低采样率下进行，从而极大地降低了系统的总运算量。

    二、填空题

    1. 答案: 贵族恒等式 (Noble Identity)
      解析: 贵族恒等式是多抽样率系统结构优化的基础。它描述了滤波器与抽取器/内插器级联时，交换两者顺序的等效关系。对于抽取器，先滤波后抽取等效于先抽取后滤波，但滤波器需变为 \(H(z^D)\)。
