    第七章 谐振器

    约 801 个字 14 张图片 预计阅读时间 3 分钟

    一、\(Q_0\)的四种求法

    1. 定义法：

    • 公式：\(\boxed{Q_0=\frac{\omega_0 L}{R}}\)

    定义回路固有品质因数\(Q_0\)： \(Q_0 = 2\pi \frac{\text{谐振电路储能}}{\text{一周期损耗}}= 2\pi \frac{\frac{1}{2}LI^2}{\frac{1}{2}I^2RT} = \frac{\omega_0 L}{R}\)

    2. \(\Delta w\)

    • 公式：\(\boxed{Q_0=\frac{w_0}{2\Delta \omega}}\)

    谐振曲线：

    在串联的LCR回路中：

    \(I = \frac{U}{Z(\omega)}\)

    其中，\(Z(\omega) = R + j\omega L + \frac{1}{j\omega C}\)

    \(= R \left[ 1 + jQ_0 \left( \frac{\omega}{\omega_0} - \frac{\omega_0}{\omega} \right) \right]\)

    \(\approx R \left( 1 + j2Q_0 \frac{\delta \omega}{\omega_0} \right)\) ( 通分可得 )

    观察谐振曲线，当\(\omega=\omega_0\)时，\(I \rightarrow I_{max}\) 此时\(Z(w)\)只存在实部。 \(2Q_0 \frac{\delta \omega}{\omega_0}=1\)时，\(|Z(\omega)|=|\sqrt{2}Z_{max}|\), 此时\(I \rightarrow 0.7 I\)

    并联LCG的推导类似，公式相同。

    3. \(\partial X(\omega_0)/\partial \omega\)（或\(\partial B(\omega_0)/\partial \omega\)）

    • 公式：
    • \(\boxed{Q_0 = \frac{1}{2} \omega_0 \frac{\partial X(\omega_0)}{\partial \omega} \Big/ R(\omega_0)}\)
    • \(\boxed{Q_0 = \frac{1}{2} \omega_0 \frac{\partial B(\omega_0)}{\partial \omega} \Big/ G(\omega_0)}\)

    由谐振曲线，谐振器阻抗实部\(R\)（或\(G\)）在\(\omega_0\)附近基本不变。阻抗虚部在\(\omega_0\)附近随\(\delta \omega\)近似线性变化。

    \(Z(\omega) = R(\omega_0) + j \left[ X(\omega_0) + \delta \omega \frac{\partial X(\omega_0)}{\partial \omega} + \ldots \right]\)， 忽略高次项， \(\approx R(\omega_0) \left[ 1 + j \frac{\delta \omega}{R(\omega_0)} \frac{\partial X(\omega_0)}{\partial \omega} \right]\)

    和刚刚的\(Z(\omega) \approx R \left( 1 + j2Q_0 \frac{\delta \omega}{\omega_0} \right)\)对比，有\(Q_0 = \frac{1}{2} \omega_0 \frac{\partial X(\omega_0)}{\partial \omega} \Big/ R(\omega_0)\)

    RCG回路同理，\(Y(\omega) \approx G(\omega_0) \left[ 1 + j \frac{\delta \omega}{G(\omega_0)} \frac{\partial B(\omega_0)}{\partial \omega} \right]\)，有\(Q_0 = \frac{1}{2} \omega_0 \frac{\partial B(\omega_0)}{\partial \omega} \Big/ G(\omega_0)\)

    4. 震荡衰减

    (不常用)

    • 公式：\(\boxed{Q_0 = \frac{1}{2} \frac{Re(\omega)}{Im(\omega)}}\)

    二、空腔谐振器

    谐振器和波导的区别：在Z方向也有振荡。设 \(z\) 方向是\(\frac{p\pi}{l}\)。

    以两端短路为例：

    谐振条件：

    Z方向谐振，有 谐振条件： \(\(\overleftarrow{Z} + \overrightarrow{Z} = 0 \iff \left\{ \begin{array}{ll} \overrightarrow{Z} = 0 & \\ \overrightarrow{Z} = jZ_z \tan(k_z l) & \text{（假设是短路推出来的）} \\ Z_z = \left\{ \begin{array}{ll} \omega \mu / k_z & \text{TE模} \\ k_z / \omega \varepsilon & \text{TM模} \end{array} \right. \end{array} \right.\)\)

    \(jZ_z \tan k_z l = 0 \rightarrow k_z = \frac{p\pi}{l}, \quad p = 1, 2, 3, \cdots\)

    谐振频率： \(k^2 = \omega^2 \mu \varepsilon = k_i^2 + k_z^2 = k_i^2 + (p\pi / l)^2\)

    TEM模同轴线，\(k_{\tau} = 0\)

    矩形波导 \(k_{\tau}^2 = \left(\frac{m\pi}{a}\right)^2 + \left(\frac{n\pi}{b}\right)^2\)

    圆波导 \(\(k_{\tau} = \left\{ \begin{array}{ll} u'_{mn} / a & \text{TE模} \\ u_{mn} / a & \text{TM模} \end{array} \right.\)\)

    谐振波长/色散条件

    可以由色散关系来求谐振波长：

    \(k^2 = \left(\frac{2\pi}{\lambda}\right)^2 = k_t^2 + \left(\frac{p\pi}{l}\right)^2\)

    TEM模同轴线

    • \(k_t=0\)，\(\lambda_0 = \frac{2l}{p}\)

    矩形波导

    \(k_i^2 = k_x^2 + k_y^2 = \left(\frac{m\pi}{a}\right)^2 + \left(\frac{n\pi}{b}\right)^2\)

    \(\lambda_0 = \frac{2}{\sqrt{(a/m)^2 + (b/n)^2 + (l/p)^2}}\)

    主模- TE101

    当\(b<a<l\)时，\(TE_{101}\)模式的谐振波长最长，是矩形空腔谐振器的主模。

    \(\boxed{\lambda_0 = \frac{2al}{\sqrt{a^2 + l^2}}}\)

    圆形波导

    \[k_t = \begin{cases} u'_{mn}/a & \text{TE模} \\ u_{mn}/a & \text{TM模} \end{cases}\]
    \[\lambda_0 = \begin{cases} 2/\sqrt{(u'_{mn}/\pi a)^2 + (p/l)^2} & \text{TE}_{mnp} \\ 2/\sqrt{(u_{mn}/\pi a)^2 + (p/l)^2} & \text{TM}_{mnp} \end{cases}\]

    场分布

    • 公式：

    • 最低模\(TE_{101}\)

    记忆方法：联想边界条件，\(E_x, E_y\) 反射系数 \(\rightarrow\) 1, \(\frac{e^{jx}-e^{-jx}}{2j}=sin(x)\)， 所以会出现\(sin\), \(H\)的表达式里从\(1- \rightarrow 1+\), 所以是\(cos\) ...

    推导：

    \(x,y\) 方向的场分布我们在波导中求过了，只需要推导\(z\)方向，方法也是类似的。

    纵向的分布由模函数的幅值\(U(z)\)、\(I(z)\)决定 $$ \begin{cases} \frac{\mathrm{d}U(z)}{\mathrm{d}z} = -jk_z Z_z I(z) \ \frac{\mathrm{d}I(z)}{\mathrm{d}z} = -jk_z Y_z U(z) \end{cases} $$ \(\(Z_z = \begin{cases} \omega \mu / k_z & \text{TE模} \\ k_z / \omega \varepsilon & \text{TM模} \end{cases}\)\)

    上两式消去\(I(z)\)得到 $$ \left( \frac{\mathrm{d}^2}{\mathrm{d}z^2} + k_z^2 \right) U(z) = 0 $$ 在边界\(z=0, l\)，切向电场为零，故得\(U(z=0, l)=0\)，在此边界条件下，其解为 $$ \begin{cases} U(z) = \sin(k_z z) \ I(z) = jY_z \cos(k_z z) \end{cases} $$ \(k_z = \frac{p\pi}{l}\)

    模式函数幅值\(U(z)\)、\(I(z)\)与模式函数相乘就得到矩形空腔谐振器横向场量。

    微扰近似下品质因数的计算

    利用初始的定义公式：\(Q_0 = 2\pi \frac{\text{周期损耗的功率} P_l}{W}\)

    \(W = W_e + W_m = \frac{1}{2} \int_V \mu |H|^2 \, dV\)（最大磁场储能）

    \(P_l = \frac{1}{2} \oint_S |J_s|^2 R_s \, dS = \frac{1}{2} R_s \oint_S |H_s|^2 \, dS\)

    由此得到

    \(Q_0 = \frac{2\pi}{T} \frac{\frac{1}{2} \int_V \mu |H|^2 \, dV}{\frac{1}{2} R_s \oint_S |H_s|^2 \, dS} = \frac{\omega \mu \int_V |H|^2 \, dV}{R_s \oint_S |H_s|^2 \, dS} = \frac{2}{\delta} \frac{\int_V |H|^2 \, dV}{\oint_S |H_s|^2 \, dS}\)

    （\(|H_s|\)是表面电流）

    （第四章： \(\frac{\omega \mu}{R_s}=\frac{2}{\delta}\)）

    如近似认为，\(|H'| \approx \frac{1}{2} |H_s|^2\)，则可得 \(Q_0 \approx \frac{1}{\delta} \frac{V}{S}\)

    • 空腔谐振器\(Q_0\)值近似与腔体体积\(V\)成正比，与其内壁表面积\(S\)成反比，与趋肤深度成反比。
    • 比值\(V/S\)越大，\(Q_0\)值越高\(\rightarrow\)球体最大。

    应用：波长计

    TE011模式场分布特点：场在圆周方向无变化，具有轴对称，谐振腔壁只有圆周方向电流，无径向电流 \(\rightarrow\) 截断径向电流， 就可以吸收非TE011的电磁波。

    谐振器与传输线耦合

    法1： \(d_{min}/\lambda_g-\omega\)

    测量\(\rho_0 \rightarrow \beta\), \(\omega_0 \rightarrow f_0\)

    原因：见圆图，\(\beta<1\)时将存在切线，此时张角最大。

    法2：功率曲线

    \(P_{\text{谐振}}=P_i-P_r\)

    \(\therefore P(w_1)=P(w_2)=\frac{1}{2}(P_i(\omega_0)-P_r(\omega_0))+P_r(\omega_0)=\frac{1}{2}(P_i(\omega_0)+P_r(\omega_0))\) 传输系数\(|\mathbf{\Gamma}(\omega)|^2=\frac{P_r}{P_i}\) \(Q_L=\frac{\omega_0}{2\Delta \omega}\) \(\(|\Gamma(\omega = \omega_0)| = \sqrt{\frac{\rho_t}{p}}, \quad \rho_0 = \frac{1 + |\Gamma(\omega = \omega_0)|}{1 - |\Gamma(\omega = \omega_0)|}, \quad \beta = \left\{ \begin{array}{ll} \rho_0, & \beta > 1 \\ 1 / \rho_0, & \beta < 1 \end{array} \right.\)\)

    判断\(\beta\)大小的微扰法：腔内插入一小损耗（如插入青草叶子），如果随微扰损耗增加，\(\omega = 0\)这一点反射功率单调增加，则\(\beta < 1\)，如果反射功率开头变小到零，后又增加，则\(\beta > 1\)。(有转折点\(\beta=1\))

    \(Q_0 = Q_L (1 + \beta), Q_e = \frac{1 + \beta}{\beta} Q_L\)

    \(\beta = \frac{Q_0}{Q_e} = \frac{Y_0'}{G_0} = \frac{Y_0}{n^2 G_0}\)
